mise en forme CSS locale

le style CSS permet la mise en forme (position, taille , couleur , police, bordure , ....) d'un element et de son contenu

un texte justifié est un texte dont le bord droit est alibné avec le bord droit de la boite englobant le texte. la propriété CSS text-align permet de gerer l'alignement (justifié, droit, gauche, centré) du texte dans un élément.

la propriete font-weight permet de mettre des elements en gras ou en très gras
la propriete font-variant permet de mettre des elements en pettites capitales
la propriete font-style permet de mettre des elements en italiques
ces propretes ne doivent pas etre confondues avec les balises .............
